In Lab 10, we learned how to deploy our web applications from our development environment to a production server. This included using git branching, server-side Linux commands, Apache configuration concepts, and file-level security using .htaccess. The goal of this lab was to move our class website into production so it can be accessed through our fully qualified domain name (FQDN), while maintaining selected folder-level security.
